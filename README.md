# 📦 Notion Image Storage

This repository is dedicated to hosting and managing images, assets, and media files used across various **Notion** pages.

## 🚀 How to Use (Embedding in Notion)

1. Upload your image files directly to the `images/` folder in this repository.
2. Copy the **Raw Link** of the uploaded image.
   - *Tip: Click on the image in GitHub, right-click the `Download` button, and select "Copy Link Address".*
3. In Notion, type `/image`, select **[Embed link]**, and paste the copied URL.
